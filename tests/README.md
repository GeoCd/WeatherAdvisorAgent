# Blogger Agent Tests

This directory contains integration tests for `Teophrastus - The Weather Advisor Agent`.

## How to Run Teophrastus Main Agent Test
You can run the test using the following command:

```bash
python -m tests.test_agent_logger_on
```

### Test Scenario

The `test_agent_logger_on.py` script test runs `Teophrastus` through a predefined sequence of user queries. This test is designed to simulate a typical conversation with the agent and ensure that it can handle the flow without errors. The script will print the agent's responses to the console for each query in the sequence.
